# SN2_UEVR
A VR first persion mod for UEVR when playing Subnautica 2

## Features include:
- Full 1st person 6DOF Motion controls
- Visible body
- Optional Forearms Only or Full Two Bone IK arms
- Articulated and animated hands
- Option to attach the UI to head movement
- Physical driving (configurable in the Subnautica 2 Config tab of the UEVR overlay)
- Physical interactions (configurable in the Subnautica 2 Config tab of the UEVR overlay)
